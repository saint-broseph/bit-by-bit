KMP (Knuth-Morris-Pratt)
Identification: Pattern matching (P in S) or finding the longest prefix which is also a suffix.
Pattern: Precompute a Pi (or LPS) array to avoid re-scanning characters on a mismatch.

Used for pattern matching:

vector<int> computeLPS(string pattern) {
    int m = pattern.length();
    vector<int> lps(m, 0);
    int len = 0, i = 1;
    while (i < m) {
        if (pattern[i] == pattern[len]) lps[i++] = ++len;
        else {
            if (len != 0) len = lps[len - 1];
            else lps[i++] = 0;
        }
    }
    return lps;
}
